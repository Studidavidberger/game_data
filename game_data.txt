 nom : clash of zombie

 categorie de jeu : clicker

 Histoire : proteger la héroine nommé Kali, des vagues de zombies ( jeu infini )

 Héroine : Kali

 Concept : cliquer frenetiquement sur des ennemis pour les tuer, et 
 faire le plus haut score.

 MVP : menu principal, barre de progression, menu, pause, powers up +degat
 boss, vie, bouclier.

 Portabilitée : Itch.io, PC, linux, et MacOS

 design : pixel, retro, nature, lumineux.

 idées : 
 
 - powerups du jeu ( +zone de degats, +degats, +bouclier, +vie, modeLSD ) ( cliquer pour le récuperer )

 - cycle jour/nuit

 - tableau des scores

 - jalons superieur a chaque boss vaincu

 - augmentation du spawn de zombies a chaque nouveau jalon

 - Pourcentage de progression, de vie, de bouclier,
 de chance qu'un boss apparaisse

 - zombies hommme, femmes, chien, poulet, esquiveur, barricadé, gargantua, cuisinier goofy ahh.. et les chaudasse du cul.
